# Mission

`skill-mastery` exists to help AI agent skills improve safely and explicitly.

Agent skills should not remain static prompt files that are written once and
forgotten. They should become maintained artifacts with clear structure,
evidence, evaluations, and reviewed improvement paths.

The mission of `skill-mastery` is to make skill creation, organization,
evaluation, evidence capture, and reviewed improvement accessible to teams and
individuals who rely on agent workflows.

In the near term, this means building the smallest useful foundation for:

- defining reusable skill artifacts
- recording privacy-safe evidence from real use
- evaluating behavior with examples and rubrics
- generating improvement proposals before trusted instructions change
- promoting skills only when they have evidence of reliability
