# Vision

`skill-mastery` aims to create a future where AI agent skills are living
artifacts that get better through use.

In that future, teams can see which skills are reliable, which ones need work,
why a skill changed, and what evidence supports its current rank. Skill
improvement is not hidden in one-off prompt edits. It is visible, evaluated,
reviewed, and rollback-friendly.

The long-term vision is a recursive improvement loop:

1. Early skills and a simple harness help create better skills.
2. Better skills help evaluate and improve the harness.
3. The improved harness helps produce safer, more capable future skills.

This is inspired by bootstrap systems where one generation of tools helps build
the next. For `skill-mastery`, each generation should remain grounded in
evidence, evaluation, and human-reviewed proposals rather than automatic
self-mutation.
